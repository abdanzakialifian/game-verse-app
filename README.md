# 🎮 GameVerse App

GameVerse is a modern game catalog app that allows users to explore games across multiple platforms with detailed information and a fast, intuitive interface.

## 🚀 Features
- 🔍 Explore games from various platforms  
- 🎮 Detailed game information  
- ⚡ Fast & responsive UI  
- 🔄 Real-time data from RAWG API  

## 🛠️ Tech Stack
- Kotlin  
- Android (Jetpack)  
- REST API (RAWG)  
- GitHub Actions (CI/CD)
